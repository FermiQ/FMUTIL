8 !> \mainpage FMUTIL: Fortran Miscellaneous UTILities
9 !! \details A Fortran library for miscellaneous utilities such as data structures
10 !! (List, Vector), root-finding for polynomials and nonlinear equations, etc.
11 !! \author Bharat Mahajan (https://www.researchgate.net/profile/Bharat_Mahajan2)
12 !! \version 0.1
13 !! \date Created: 02/12/2020
14 !! \copyright Copyright 2020 Bharat Mahajan
15 !! \section sec Introduction
16 !! FMUTil is a modern object-oriented Fortran library that provides (as of now):
17 !! - Two data structures
18 !! + Vector, similar to C++ STL Vectors
19 !! + List, similar to Python List
20 !! - Root-finding algorithm for polynomials by solving associated eigenvalue
21 !! problem
22 !! - Root-finding of a nonlinear equation using Brent's algorithm
23 !! - A binary search method for sorted arrays
24 !!
25 !! \section usagesec How to Use
26 !! + For Vector data structure of any type, define the type of the Vector
27 !! element and the assignment operation as follows.
28 !!
29 !! \code{.f90}
30 !! module TestModule
31 !! use FMUtil
32 !! type, extends(VecElem) :: relem
33 !! real :: rdata
34 !! contains
35 !! procedure :: AssignVecElem => assign_relem
36 !! end type relem
37 !!
38 !! contains
39 !!
40 !! subroutine assign_relem(lhs, rhs)
41 !! implicit none
42 !! class(relem), intent(out) :: lhs
43 !! class(VecElem), intent(in) :: rhs
44 !! select type (rhs)
45 !! class is (relem)
46 !! lhs%rdata = rhs%rdata
47 !! end select
48 !! end subroutine assign_relem
49 !! end module TestModule
50 !! \endcode
51 !!
52 !! + Then, use Vector using
53 !!
54 !! \code{.f90}
55 !! program Test
56 !! use FMUtil
57 !! use TestModule
58 !!
59 !! integer :: ctr
60 !! type(Vector) :: vec
61 !! type(relem) :: rd1, rd2
62 !!
63 !! rd1%rdata = 3.1416
64 !! ctr = vec%PushBack(rd1)
65 !! rd1%rdata = 2.718
66 !! call vec%Insert(2,[rd1, rd1])
67 !!
68 !! do ctr = 1, vec%Size()
69 !! rd2 = vec%ElemAt(ctr)
70 !! print *, 'int(', ctr, ')=', rd2%rdata
71 !! end do
72 !! end program Test
73 !! \endcode
74 !!

112 !! \section Installation
113 !! FMUTIL is tested with Intel Fortran Compiler and MinGW-W64 gfortran. Doxyfile is
114 !! provided for generating extensive API documentation using Doxygen. FLINT has no
115 !! dependency on any other library. The CMakeLists file is provided along with
116 !! additional CMake modules for compiler options and a Find module for finding and
117 !! linking Intel MKL is also provided. These files can be used to auto generate
118 !! Visual Studio projects or makefiles on Windows and Linux using CMake.
119 !! A find module for FMUTIL is provided that generates CMake config files for
120 !! easily linking FMUTIL using the find_package() command. The steps to link FMUTIL
121 !! in cmake-based projects are:
122 !!
123 !! + In cmake GUI or command-line, set FMUTIL_INSTALL_LIB_DIR to the desired directory,
124 !! where the compiled library, modules, and cmake config files will be installed.
125 !! + In cmake GUI or command-line, set FMUTIL_INSTALL_BIN_DIR to the desired directory,
126 !! where the compiled test executables will be installed.
127 !! + In your project CMakeLists.txt, insert
128 !!
129 !! \code
130 !! find_package(FMUTIL REQUIRED 0.1 CONFIG
131 !! PATHS "<SAME_PATH_AS_IN_FMUTIL_INSTALL_LIB_DIR>" NO_CMAKE_PACKAGE_REGISTRY)
132 !! target_link_libraries(<YOUR_TARGET_NAME> FMUTIL::FMUTIL)
133 !! \endcode
